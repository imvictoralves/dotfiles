# 1. Record decisions

Date: 2022-03-04

## Context

I want to record decisions I make in my workstation.

## Decision

Use [Architecture Decision Records][1]. To practice the habit of writing these
decision records.

## Consequences

See [Michael Nygard's article][1]. For a lightweight ADR tool set, see Nat
Pryce's [adr-tools][2].
